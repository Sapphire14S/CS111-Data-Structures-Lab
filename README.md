## Laboratory Overview

This repository contains practical assignments and implementations related to arrays, linked lists, stacks, queues, trees, heaps, graphs, hashing, sorting algorithms, recursion, and advanced data structure concepts using the C programming language.

---

## Labs Overview

| Lab | Concepts Covered |
|-----|------------------|
| Lab&nbsp;01 | Array operations, subset checking, duplicate removal, factorial computation, matrix neighbor traversal, array reordering problems |
| Lab&nbsp;02 | Array reversal, frequency counting, merging sorted arrays, string manipulation, matrix path problems, filtering and arrangement techniques |
| Lab&nbsp;03 | Singly linked lists, linked list traversal, insertion operations, recursive node counting, sorted linked list insertion |
| Lab&nbsp;04 | Circular linked lists, doubly linked lists, node deletion, list reversal, searching operations, duplicate removal in linked lists |
| Lab&nbsp;05 | Stack implementation using linked lists, balanced parentheses checking, infix-to-postfix/prefix conversion, postfix evaluation |
| Lab&nbsp;06 | Queue implementation using circular arrays and linked lists, circular doubly linked lists, queue implementation using stacks |
| Lab&nbsp;07 | Binary trees, tree traversals, binary tree construction, half-node counting, Lowest Common Ancestor (LCA) in BST |
| Lab&nbsp;08 | Array searching problems, repeated element detection, comparison of sorting algorithms, pair-sum problems |
| Lab&nbsp;09 | Binary Search Trees (BST), BST validation, BST comparison, range queries in BST |
| Lab&nbsp;10 | Menu-driven BST implementation, insertion, deletion, traversal, searching, merging binary search trees |
| Lab&nbsp;11 | Heap operations, kth largest/smallest element problems, Min Heap implementation, Heap Sort |
| Lab&nbsp;12 | AVL trees, graph traversal algorithms (DFS/BFS), graph isomorphism, hashing and collision handling |
